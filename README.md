# Job Portal Automation Framework

## Overview
This project demonstrates an end-to-end UI automation framework
for a job portal application using Selenium and Python.

## Features
- Page Object Model (POM)
- Explicit waits
- Profile update workflow
- GitHub-safe demo configuration

## Tech Stack
- Python
- Selenium
- Pytest

## Disclaimer
This project does not automate any real production system.
All URLs, locators, and credentials are mock representations
for learning and demonstration purposes only.
